# Модель предметной области

Книга (Book) — основной объект учёта.
Читатель (Reader) — субъект, получающий книги.
Выдача (Loan) — событие связывания книги и читателя во времени.
Репозиторий (LibraryRepository) — интерфейс взаимодействия с хранилищем данных.

UseCase — реализация пользовательского сценария, оперирующего сущностями.
