# Бизнес-правила

1. Каждая книга имеет уникальный идентификатор и может находиться только в одном состоянии: доступна или выдана.
2. Выдача книги возможна только зарегистрированному читателю.
3. Нельзя списать книгу, если она выдана.
4. Возврат книги переводит её состояние в “доступна” и закрывает активный займ.
5. Репозиторий обеспечивает согласованность данных о книгах, читателях и займах.
6. Добавление книг возможно только пользователем с ролью “Библиотекарь”.
7. Все операции фиксируются с указанием даты и времени (в будущем — для отчётности).
