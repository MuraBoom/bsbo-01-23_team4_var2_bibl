# Словарь терминов

| Термин                               | Определение                                                                                              |
| ------------------------------------ | -------------------------------------------------------------------------------------------------------- |
| **Книга (Book)**                     | Экземпляр произведения, хранящийся в библиотеке, имеющий уникальный идентификатор и атрибут доступности. |
| **Читатель (Reader)**                | Зарегистрированный пользователь библиотеки, имеющий право брать книги.                                   |
| **Выдача (Loan)**                    | Запись, фиксирующая факт получения книги читателем, с датой выдачи и возможной датой возврата.           |
| **Библиотекарь (Librarian)**         | Пользователь системы, выполняющий учёт фонда, выдачу и возврат книг.                                     |
| **Каталог (Catalog)**                | Совокупность всех книг, зарегистрированных в библиотечной системе.                                       |
| **Репозиторий (Repository)**         | Программный компонент, управляющий доступом к данным (в памяти, базе или файле).                         |
| **UseCase (Сценарий использования)** | Завершённая пользовательская операция в рамках системы (например, “Выдать книгу”).                       |
| **Прецедент**                        | Конкретный случай использования системы, описанный в виде диаграммы последовательности.                  |
